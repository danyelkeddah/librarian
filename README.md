# The Librarian

A meta-skill for private-first distribution of skills, agents, and prompts across your agents, devices, and teams. It maintains a single catalog (`librarian.yaml`) that points to where everything lives — local file paths or GitHub URLs — and pulls items on demand. The Librarian handles fetching, syncing, dependency resolution, and pushing changes back to the source.
